3.3. Basic Selector types

  1. Selector allows us to tell the browser which element/tag on the page we want to style
  2. CSS selectors are used to "target/find/select" HTML elements based on element/tag name, id, class, attribute, and more
  3. Selector may target: Single element, group of elements, .class, #id, any specific conditional element
  4. Selectors are used to declare which of the markup elements a style applies to, Selector indicates the HTML element you want to style

3.9. conditional descendent Selector

  • Highly specific selector that target elements based on their location within other elements
  • You can use this selector when you need to select an element that is the descendant/inside of another element
  • You want to apply a style rule to a particular element only when it lies inside a particular element
  • White space is used to denote hierarchy of element (parent nested child)
