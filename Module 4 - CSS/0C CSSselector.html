css selector

css selector is a pattern used to select the element(s) you want to style.

we can divide css selectors into five categories:

  • simple selectors (select elements based on name, id, class)
  • combinator selectors (select elements based on a specific relationship between them)
  • pseudo-class selectors (select elements based on a certain state)
  • pseudo-elements selectors (select and style a part of an element)
  • attribute selectors (select elements based on an attribute or attribute value)

simple selectors

simple selectors are based on the following:

  • element name
  • id
  • class
  • universal
  • attribute
  • attribute value

element name

selects all elements with the given tag name.

example:

        
            p {
                color: red;
            }

id

selects an element based on the value of its id attribute.

example:

        
            #para1 {
                text-align: center;
                color: red;
            }

class

selects all elements with the given class.

example:

        
            .center {
                text-align: center;
                color: red;
            }

universal

selects all elements in the document.

example:

        
            * {
                text-align: center;
                color: red;
            }

attribute

selects all elements with the given attribute.

example:

        
            a[target] {
                background-color: yellow;
            }

attribute value

selects all elements with the given attribute and value.

example:

        
            a[target="_blank"] {
                background-color: yellow;
            }

combinator selectors

combinator selectors are used to combine multiple selectors into more specific selectors.

there are four different combinators in css:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

descendant selector

selects all elements that are descendants of a given ancestor.

example:

        
            div p {
                background-color: yellow;
            }

child selector

selects all elements that are the children of a given element.

example:

        
            div > p {
                background-color: yellow;
            }

adjacent sibling selector

selects all elements that are the adjacent siblings of a given element.

example:

        
            div + p {
                background-color: yellow;
            }

general sibling selector

selects all elements that are the siblings of a given element.

example:

        
            div ~ p {
                background-color: yellow;
            }

pseudo-class selectors

pseudo-class selectors are used to define the special state of an element.

for example, it can be used to:

  • style an element when a user mouses over it
  • style visited and unvisited links differently
  • style an element when it gets focus

there are many pseudo-class selectors available in css:

  • :active
  • :checked
  • :disabled
  • :empty
  • :enabled
  • :first-child
  • :first-of-type
  • :focus
  • :hover
  • :in-range
  • :invalid
  • :lang(language)
  • :last-child
  • :last-of-type
  • :link
  • :not(selector)
  • :nth-child(n)
  • :nth-last-child(n)
  • :nth-last-of-type(n)
  • :nth-of-type(n)
  • :only-of-type
  • :only-child
  • :optional
  • :out-of-range
  • :read-only
  • :read-write
  • :required
  • :root
  • :target
  • :valid
  • :visited

pseudo-elements selectors

pseudo-elements selectors are used to style specified parts of an element.

for example, it can be used to:

  • style the first letter, or line, of an element
  • insert content before, or after, the content of an element

there are many pseudo-elements selectors available in css:

  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection

attribute selectors

attribute selectors are used to select elements based on their attribute values.

there are many attribute selectors available in css:

  • [attribute]
  • [attribute=value]
  • [attribute~=value]
  • [attribute|=value]
  • [attribute^=value]
  • [attribute$=value]
  • [attribute*=value]

css selector precedence

if there are two or more conflicting css rules that point to the same element, the browser follows some rules to determine which one is most specific and therefore wins out.

the following list of selector types increases by specificity:

  • type selectors (e.g., h1) and pseudo-elements (e.g., ::before).
  • class selectors (e.g., .example), attributes selectors (e.g., [type="radio"]) and pseudo-classes (e.g., :hover).
  • id selectors (e.g., #example).

universal selector (*) has no specificity value (0, 0, 0, 0).

combinators (+, >, ~, ' ', ||) and negation pseudo-class (:not()) have no specificity value (0, 0, 0, 0) but are added where appropriate.

examples:

        
            /* specificity: 0, 0, 0, 1 */
            :link {
                color: red;
            }
            /* specificity: 0, 0, 1, 0 */
            ul li {
                color: blue;
            }
            /* specificity: 0, 1, 0, 0 */
            #myid {
                color: green;
            }
            /* specificity: 1, 0, 0, 0 */
            h1 {
                color: yellow;
            }
            /* specificity: 0, 0, 0, 2 */
            ul ol+li {
                color: black;
            }
            /* specificity: 0, 0, 0, 1 */
            li.red.level {
                color: silver;
            }
            /* specificity: 0, 0, 0, 3 */
            style[type="text/css"] {
                color: white;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-child(2n+1) {
                color: fuchsia;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-last-child(2n+1) {
                color: aqua;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-of-type(2n+1) {
                color: gray;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-last-of-type(2n+1) {
                color: lime;
            }
            /* specificity: 0, 0, 0, 1 */
            p:first-child {
                color: maroon;
            }
            /* specificity: 0, 0, 0, 1 */
            p:last-child {
                color: olive;
            }
            /* specificity: 0, 0, 0, 1 */
            p:first-of-type {
                color: navy;
            }
            /* specificity: 0, 0, 0, 1 */
            p:last-of-type {
                color: teal;
            }
            /* specificity: 0, 0, 0, 1 */
            p:only-child {
                color: purple;
            }
            /* specificity: 0,
            0, 0, 1 */
            p:only-of-type {
                color: orange;
            }
            /* specificity: 0, 0, 0, 1 */
            p:empty {
                color: red;
            }
            /* specificity: 0, 0, 0, 1 */
            p:lang(it) {
                color: blue;
            }
            /* specificity: 0, 0, 0, 1 */
            p:root {
                color: green;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-child(2n+1):not(:first-child) {
                color: yellow;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-child(2n+1):not(:last-child) {
                color: black;
            }
            /* specificity: 0, 0, 0, 1 */
            p:nth-child(2n+1):not(:only-child) {
                color: silver;
            }

css selector list

you can group selectors to apply the same styles to several selectors.

to group selectors, separate each selector with a comma.

example:

        
            h1, h2, p {
                text-align: center;
                color: red;
            }
