Colours

Colours are specified using predefined colour names, or RGB, HEX, HSL, RGBA, HSLA values.

Colour names are case-insensitive.

Colour values can be specified in the following ways:

  • RGB (Red, Green, Blue) - Each of the three colours is specified using a value between 0 and 255.
  • HEX (Hexadecimal) - A six-digit code preceded by a hash (#) which specifies the amount of red, green and blue in a colour.
  • HSL (Hue, Saturation, Lightness) - Each of the three colours is specified as a percentage between 0% and 100%.
  • RGBA (Red, Green, Blue, Alpha) - Same as RGB, but with an additional value which specifies the opacity of the colour. The alpha value can be a number between 0.0 (fully transparent) and 1.0 (fully opaque).
  • HSLA (Hue, Saturation, Lightness, Alpha) - Same as HSL, but with an additional value which specifies the opacity of the colour. The alpha value can be a number between 0.0 (fully transparent) and 1.0 (fully opaque).

  • Orange: orange
  • Pink: pink
  • Purple: purple
  • Brown: brown
